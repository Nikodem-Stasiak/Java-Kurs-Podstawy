# ☕ Java - Moja Droga do Inżyniera

Repozytorium zawierające moje postępy, ćwiczenia i notatki z nauki języka Java. Dokumentuję tutaj fundamenty programowania obiektowego oraz logiki systemów.

## 📚 Czego się już nauczyłem?
* **Podstawy składni:** Zmienne (`int`, `String`, `boolean`), operatory matematyczne i logiczne.
* **Instrukcje sterujące:** Pętle (`while`, `for`, `for-each`), instrukcje warunkowe (`if-else`, `switch`).
* **Programowanie Obiektowe (OOP):** Klasy, obiekty, konstruktory (w tym przeciążanie), hermetyzacja (prywatne pola i gettery).
* **Obsługa danych:** Praca z klasą `Scanner` (input od użytkownika), listy dynamiczne (`ArrayList`).
* **Solidne fundamenty:** Obsługa wyjątków (`try-catch`), system kontroli wersji (Git & GitHub).

## 🛠️ Narzędzia
* **IDE:** IntelliJ IDEA
* **Język:** Java 21+
* **System kontroli wersji:** Git
